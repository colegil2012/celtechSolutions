# Celtech Solutions — Landing Site

A small Spring Boot 4 / Thymeleaf landing page for **Celtech Solutions**.

## Stack

- **Java 21**, **Spring Boot 4** (web, security, validation, thymeleaf, data-mongodb)
- **Thymeleaf** server-rendered views with shared layout fragments
- Hand-rolled CSS organised into `base/`, `layout/`, `components/`, `pages/`
- **AOS** for scroll-triggered animations
- **GSAP** for the hero intro animation
- **Alpine.js** for tiny reactive bits (mobile menu, form state)
- **Lucide** for icons

## Run

```bash
./mvnw spring-boot:run
```
